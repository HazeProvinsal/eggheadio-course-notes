<TimeStamp start="2:45" end="3:00">

```jsx
const sendMessage = async evt => {
    evt.preventDefault()
    const content = message.current.value
    await supabase
    .from('message')
    .insert([
        {content, user_id: session.user.id}
    ])
}
```
